# student-management-system-python-mysql
Developed a Student Management System using Python and MySQL implementing CRUD operations, database connectivity, record management, and SQL query execution. Designed to efficiently store, retrieve, update, and manage student information.
# Student Management System

A Python and MySQL based Student Management System that performs CRUD operations on student records.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

## Technologies Used

- Python
- MySQL
- mysql-connector-python

## Project Structure

StudentManagementSystem/
│
├── main.py
├── db.py
├── database.sql
├── requirements.txt
└── README.md

## How to Run

1. Install MySQL
2. Create database using database.sql
3. Update MySQL credentials in db.py
4. Install dependencies

pip install mysql-connector-python

5. Run

python main.py
